## ChartGPT
ChartGPT: AI based UML tool ChartGPT

## Requirements
1. Node Version >= 18
2. OpenAI API Key

## Installation

1. Clone the Repo

2. Open `.env.local` file and enter OPENAI_API_KEY

2. `cd` into the folder
```sh
cd chartGPT
```

3. Install all deps with yarn
```sh
yarn
```

4. Run in development mode
```sh
yarn dev
```
